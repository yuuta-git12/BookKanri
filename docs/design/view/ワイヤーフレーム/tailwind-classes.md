# Tailwind CSS クラス一覧

C-01・U-01 ワイヤーフレームで使用しているTailwind CSSクラスをカテゴリ別にまとめたもの。

---

## 1. レイアウト・配置


| クラス                | 説明                               |
| ------------------ | -------------------------------- |
| `flex`             | 要素をフレックスコンテナにする                  |
| `items-center`     | 交差軸方向（縦）に中央揃え                    |
| `items-start`      | 交差軸方向（縦）に上揃え                     |
| `justify-center`   | 主軸方向（横）に中央揃え                     |
| `justify-between`  | 主軸方向（横）に両端揃え（間隔均等）               |
| `grid`             | 要素をグリッドコンテナにする                   |
| `grid-cols-2`      | グリッドを2列に分割する                     |
| `relative`         | 位置の基準点を自身に設定（`absolute` の親要素向け）  |
| `absolute`         | 最も近い `relative` 親を基準に絶対位置指定      |
| `right-3`          | 右から `0.75rem (12px)` の位置に配置      |
| `top-1/2`          | 上から50%の位置に配置                     |
| `-translate-y-1/2` | Y軸方向に自身の高さの50%だけ上にずらす（垂直中央揃えに使用） |
| `overflow-hidden`  | はみ出したコンテンツを非表示にする（角丸と組み合わせて使用）   |
| `divide-y`         | 子要素間に水平方向の区切り線を引く                |
| `divide-gray-100`  | `divide-y` の線色を `gray-100` に設定   |

## 2. サイズ・余白

### 幅・高さ


| クラス        | 説明                                     |
| ---------- | -------------------------------------- |
| `w-full`   | 幅を親要素の100%にする                          |
| `w-52`     | 幅を `13rem (208px)` に固定                 |
| `w-80`     | 幅を `20rem (320px)` に固定                 |
| `w-100`    | 幅を `25rem (400px)` に固定（Tailwind v3.3+） |
| `w-3.5`    | 幅を `0.875rem (14px)` に固定（アイコン用）        |
| `w-6`      | 幅を `1.5rem (24px)` に固定（アバター用）          |
| `w-7`      | 幅を `1.75rem (28px)` に固定（ページネーションボタン用）  |
| `h-7`      | 高さを `1.75rem (28px)` に固定               |
| `h-6`      | 高さを `1.5rem (24px)` に固定                |
| `h-11`     | 高さを `2.75rem (44px)` に固定（ナビバー用）        |
| `h-3.5`    | 高さを `0.875rem (14px)` に固定              |
| `min-h-64` | 最小高さを `16rem (256px)` に設定              |


### パディング


| クラス      | 説明                                        |
| -------- | ----------------------------------------- |
| `p-5`    | 全方向に `1.25rem (20px)` のパディング              |
| `p-8`    | 全方向に `2rem (32px)` のパディング                 |
| `px-1`   | 左右に `0.25rem (4px)` のパディング                |
| `px-2`   | 左右に `0.5rem (8px)` のパディング                 |
| `px-3`   | 左右に `0.75rem (12px)` のパディング               |
| `px-4`   | 左右に `1rem (16px)` のパディング                  |
| `px-5`   | 左右に `1.25rem (20px)` のパディング               |
| `px-8`   | 左右に `2rem (32px)` のパディング                  |
| `py-1.5` | 上下に `0.375rem (6px)` のパディング               |
| `py-2`   | 上下に `0.5rem (8px)` のパディング                 |
| `py-2.5` | 上下に `0.625rem (10px)` のパディング              |
| `py-3`   | 上下に `0.75rem (12px)` のパディング               |
| `py-16`  | 上下に `4rem (64px)` のパディング                  |
| `pr-9`   | 右に `2.25rem (36px)` のパディング（表示ボタン分のスペース確保） |


### マージン


| クラス      | 説明                        |
| -------- | ------------------------- |
| `mb-1`   | 下に `0.25rem (4px)` のマージン  |
| `mb-2`   | 下に `0.5rem (8px)` のマージン   |
| `mb-3`   | 下に `0.75rem (12px)` のマージン |
| `mb-4`   | 下に `1rem (16px)` のマージン    |
| `mb-6`   | 下に `1.5rem (24px)` のマージン  |
| `mt-0.5` | 上に `0.125rem (2px)` のマージン |
| `mt-1`   | 上に `0.25rem (4px)` のマージン  |
| `mt-2`   | 上に `0.5rem (8px)` のマージン   |
| `mr-2`   | 右に `0.5rem (8px)` のマージン   |


### ギャップ・スペース


| クラス           | 説明                                    |
| ------------- | ------------------------------------- |
| `gap-1.5`     | グリッド・フレックスの子要素間を `0.375rem (6px)` 空ける |
| `gap-2`       | グリッド・フレックスの子要素間を `0.5rem (8px)` 空ける   |
| `gap-4`       | グリッド・フレックスの子要素間を `1rem (16px)` 空ける    |
| `gap-5`       | グリッド・フレックスの子要素間を `1.25rem (20px)` 空ける |
| `space-y-4`   | 縦方向の子要素間に `1rem (16px)` のマージンを設定      |
| `space-y-12`  | 縦方向の子要素間に `3rem (48px)` のマージンを設定      |
| `space-x-1`   | 横方向の子要素間に `0.25rem (4px)` のマージンを設定    |
| `space-y-1.5` | 縦方向の子要素間に `0.375rem (6px)` のマージンを設定   |

## 4. 色・背景

### 背景色


| クラス             | 説明                                 |
| --------------- | ---------------------------------- |
| `bg-white`      | 背景を白（`#ffffff`）にする                 |
| `bg-gray-50`    | 背景をごく薄いグレー（`#f9fafb`）にする           |
| `bg-gray-100`   | 背景を薄いグレー（`#f3f4f6`）にする             |
| `bg-slate-800`  | 背景を濃いネイビー（`#1e293b`）にする（ナビバー・ボタン用） |
| `bg-blue-500`   | 背景を青（`#3b82f6`）にする（アバター用）          |
| `bg-green-100`  | 背景を薄い緑（`#dcfce7`）にする（バッジ用）         |
| `bg-blue-100`   | 背景を薄い青（`#dbeafe`）にする（バッジ用）         |
| `bg-yellow-100` | 背景を薄い黄（`#fef9c3`）にする（バッジ用）         |
| `bg-red-50`     | 背景をごく薄い赤（`#fef2f2`）にする（エラーバナー用）    |


### テキスト色


| クラス               | 説明                           |
| ----------------- | ---------------------------- |
| `text-white`      | テキストを白にする                    |
| `text-gray-300`   | テキストを薄いグレーにする（未評価の星など）       |
| `text-gray-400`   | テキストをグレーにする（プレースホルダー・補足テキスト） |
| `text-gray-500`   | テキストをやや濃いグレーにする（ラベル・操作ボタン）   |
| `text-gray-600`   | テキストをさらに濃いグレーにする（テーブルセル）     |
| `text-gray-700`   | テキストを濃いグレーにする                |
| `text-gray-800`   | テキストをより濃いグレーにする（テーブル書籍名）     |
| `text-gray-900`   | テキストをほぼ黒にする                  |
| `text-slate-100`  | テキストを明るいスレート色にする（ナビ内テキスト）    |
| `text-slate-200`  | テキストをやや明るいスレート色にする           |
| `text-slate-800`  | テキストを濃いネイビーにする（見出し）          |
| `text-green-800`  | テキストを濃い緑にする（バッジ）             |
| `text-blue-800`   | テキストを濃い青にする（バッジ）             |
| `text-yellow-800` | テキストを濃い黄にする（バッジ）             |
| `text-yellow-400` | テキストを明るい黄にする（星評価）            |
| `text-red-400`    | テキストを明るい赤にする（必須マーク・アイコン）     |
| `text-red-500`    | テキストを赤にする（フィールドエラーメッセージ）     |
| `text-red-600`    | テキストをやや濃い赤にする（エラーバナー）        |

## 5. ボーダー・角丸・シャドウ

### ボーダー


| クラス                | 説明                               |
| ------------------ | -------------------------------- |
| `border`           | 全方向に `1px` のボーダーを表示              |
| `border-b`         | 下方向のみ `1px` のボーダーを表示             |
| `border-r-2`       | 右方向のみ `2px` のボーダーを表示             |
| `border-gray-100`  | ボーダー色を `gray-100` に設定            |
| `border-gray-200`  | ボーダー色を `gray-200` に設定            |
| `border-red-200`   | ボーダー色を `red-200` に設定（エラーバナー枠）    |
| `border-red-300`   | ボーダー色を `red-300` に設定（エラー入力欄）     |
| `border-slate-800` | ボーダー色をネイビーに設定（サイドバーアクティブインジケーター） |


### 角丸


| クラス            | 説明                              |
| -------------- | ------------------------------- |
| `rounded`      | `border-radius: 0.25rem (4px)`  |
| `rounded-lg`   | `border-radius: 0.5rem (8px)`   |
| `rounded-xl`   | `border-radius: 0.75rem (12px)` |
| `rounded-full` | 完全な円形にする（アバター用）                 |


### シャドウ


| クラス         | 説明                         |
| ----------- | -------------------------- |
| `shadow-sm` | 小さいドロップシャドウを付ける（カード・テーブル用） |

## 6. インタラクション


| クラス                  | 説明                             |
| -------------------- | ------------------------------ |
| `hover:bg-gray-50`   | ホバー時に背景を `gray-50` にする         |
| `hover:bg-slate-700` | ホバー時に背景を `slate-700` にする（ボタン用） |
| `cursor-pointer`     | カーソルをポインター（手）にする               |
| `select-none`        | テキスト選択を無効にする（ボタンラベル用）          |

## 7. カスタムクラス（`@apply` 定義）

`<style>` タグ内で定義された再利用クラス。


| クラス名                   | 適用内容                                                                                    | 用途                 |
| ---------------------- | --------------------------------------------------------------------------------------- | ------------------ |
| `.badge`               | `inline-block text-xs font-medium px-2 py-0.5 rounded-full`                             | ステータスバッジ（読了・読書中など） |
| `.action-btn`          | `text-xs border border-gray-200 rounded px-2 py-0.5 text-gray-500 hover:bg-gray-50`     | テーブル内の操作ボタン（詳細・編集） |
| `.sidebar-item`        | `flex items-center gap-2 px-4 py-2 text-sm text-gray-500 cursor-pointer hover:bg-white` | サイドバーのメニュー項目       |
| `.sidebar-item.active` | `bg-white text-gray-900 font-medium border-r-2 border-slate-800`                        | サイドバーのアクティブ項目      |
| `.form-label`          | `block text-xs font-medium text-gray-500 mb-1`                                          | フォームのラベル           |
| `.form-input`          | `w-full border border-gray-200 rounded-lg px-3 py-1.5 text-sm text-gray-400 bg-white`   | 通常状態のフォーム入力欄       |
| `.form-input-error`    | `w-full border border-red-300 rounded-lg px-3 py-1.5 text-sm text-gray-400 bg-white`    | エラー状態のフォーム入力欄      |
| `.form-select`         | `w-full border border-gray-200 rounded-lg px-3 py-1.5 text-sm bg-white`                 | フォームのセレクトボックス      |
