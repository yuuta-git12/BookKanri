### 利用ユーザー向け画面


| 画面ID | 画面名            | 概要                         | 遷移元                        | 遷移先                          |
| ---- | -------------- | -------------------------- | -------------------------- | ---------------------------- |
| U-01 | 書籍一覧画面         | 自分が登録した書籍を一覧表示する。検索も本画面で行う | ログイン成功・登録完了・編集完了・削除完了・検索実行 | U-02 / U-04 / U-05           |
| U-02 | 書籍詳細画面         | 選択した書籍の詳細情報を表示する           | U-01                       | U-05                         |
| U-03 | 書籍登録画面（ISBN入力） | ISBNを入力し外部APIから書籍情報を取得する   | U-01                       | U-04（ISBN あり）/ U-06（ISBN なし） |
| U-04 | 書籍情報確認・登録画面    | APIから取得した書籍情報を確認して登録する     | U-03                       | U-01                         |
| U-05 | 書籍情報編集画面       | 登録済み書籍の情報を編集する             | U-02                       | U-02                         |
| U-06 | 手動入力フォーム       | ISBNがない書籍を手動で入力して登録する      | U-03                       | U-04                         |


> **備考**  
> 削除確認はフロー図上では判断（ダイヤモンド）として定義されているため、独立した画面ではなくモーダルダイアログで実装する想定。

### 管理者向け画面


| 画面ID | 画面名         | 概要                               | 遷移元    | 遷移先         |
| ---- | ----------- | -------------------------------- | ------ | ----------- |
| A-01 | 管理メニュー画面    | ユーザー管理・書籍データ管理へ遷移するメニュー          | ログイン成功 | A-02 / A-05 |
| A-02 | ユーザー一覧画面    | 全利用ユーザーを一覧表示する                   | A-01   | A-03 / A-04 |
| A-03 | ユーザー登録フォーム  | 新規利用ユーザーを登録する                    | A-02   | A-02        |
| A-04 | ユーザー情報編集画面  | 利用ユーザーの情報を編集する                   | A-02   | A-02        |
| A-05 | 全ユーザー書籍一覧画面 | 全利用ユーザーの書籍データを一覧表示する。閲覧・編集・削除を行う | A-01   | A-01        |


> **備考**  
> 管理者による書籍の編集・削除は U-05・削除確認モーダルを流用する想定。  
> ユーザーの削除確認もモーダルダイアログで実装する想定。

### C-01 ログイン画面

**表示項目**


| No  | 項目名     | 内容              | 備考  |
| --- | ------- | --------------- | --- |
| 1   | メールアドレス | テキスト入力欄         |     |
| 2   | パスワード   | パスワード入力欄（マスク表示） |     |
| 3   | ログインボタン | 認証を実行する         |     |


**操作**


| No  | 操作名  | アクション | 遷移先                        |
| --- | ---- | ----- | -------------------------- |
| 1   | ログイン | 認証成功  | 利用ユーザー → U-01 / 管理者 → A-01 |
| 2   | ログイン | 認証失敗  | C-01（エラーメッセージ表示）           |


**バリデーション**


| No  | 項目名     | ルール      | エラーメッセージ                                                                               |
| --- | ------- | -------- | -------------------------------------------------------------------------------------- |
| 1   | メールアドレス | 必須・メール形式 | 空の場合：メールアドレスを入力してください 未登録のメールアドレスの場合：未登録のメールアドレスです                                     |
| 2   | パスワード   | 必須       | 空の場合：パスワードを入力してください パスワードが違う場合：パスワードが違います<br>メールアドレスとパスワードの組が違う場合：メールアドレスまたはパスワードが違います |
